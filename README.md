Booking Project with PHP 8.2.13 && Laravel Framework 10.23.1

Used packages:

    for user - laravel/breeze
    for admin - tcg/voyager
    for validation - respect/validation

Commands:

    To start
    
        make start
        make migration
        make seed
    
    To up:
    
        make up
    
    To stop:
    
        make stop
    
    To create admin:
    
        sail artisan voyager:admin your@email.com --create
